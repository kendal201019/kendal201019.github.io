Description

Welcome to my GitHub Pages portfolio for my CS 499 Final Project. This page hosts several important materials related to my capstone project, which demonstrates my growth in the key areas of software engineering, algorithms, and databases.

For this final project, I developed and enhanced a tenant screening application. The application evaluates rental applicants based on key financial criteria such as credit score, income-to-rent ratio, and employment length. This project, originally written as a Python script, was transformed into a full-stack web application using Angular for the frontend and Node.js with MongoDB for the backend.

The project focuses on three key areas of improvement:

  • Enhancing the software engineering aspects by transitioning the application from a standalone Python script to a modular, scalable full-stack solution.
  • Optimizing the algorithms to ensure efficient handling of multiple applicants and large datasets, ensuring scalability without performance degradation.
  • Integrating MongoDB for persistent data storage, enabling the system to store, retrieve, and analyze tenant applications securely.
